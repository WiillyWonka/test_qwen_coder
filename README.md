# AI News Agent - Бот для новостей об ИИ в Telegram

Этот проект представляет собой AI-агента на базе LangChain, который:
1. 📰 Собирает последние новости в области искусственного интеллекта из RSS-лент
2. 🤖 Суммаризирует их с помощью языковой модели
3. 📱 Отправляет сводку вам в Telegram

## Установка

### 1. Установите зависимости

```bash
pip install langchain langchain-community langchain-core python-telegram-bot feedparser aiohttp
```

### 2. Получите токен Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Создайте нового бота командой `/newbot`
3. Следуйте инструкциям и скопируйте полученный токен
4. Установите переменную окружения:

```bash
export TELEGRAM_BOT_TOKEN='ваш_токен'
```

### 3. (Опционально) Настройте языковую модель

По умолчанию используется заглушка для тестирования. Для реальной суммаризации подключите одну из LLM:

#### OpenAI GPT
```bash
export OPENAI_API_KEY='ваш_ключ_openai'
```
В коде раскомментируйте блок "ВАРИАНТ 1" в функции `create_llm()`.

#### Anthropic Claude
```bash
export ANTHROPIC_API_KEY='ваш_ключ_anthropic'
```
В коде раскомментируйте блок "ВАРИАНТ 2" в функции `create_llm()`.

#### Ollama (локальная модель)
Установите [Ollama](https://ollama.ai/) и запустите:
```bash
ollama run llama3
```
В коде раскомментируйте блок "ВАРИАНТ 3" в функции `create_llm()`.

## Запуск

```bash
python ai_news_agent.py
```

## Команды бота

- `/start` - Приветствие и информация о боте
- `/news` - Получить последние новости об ИИ вручную
- `/subscribe` - Подписаться на ежедневные новости (приходят в 9:00)
- `/unsubscribe` - Отписаться от новостей
- `/help` - Справка

## Как это работает

1. **Сбор новостей**: Агент парсит RSS-ленты популярных источников:
   - Artificial Intelligence News
   - Google AI Blog
   - OpenAI Blog
   - VentureBeat AI
   - TechCrunch AI

2. **Суммаризация**: Каждая новость обрабатывается через LangChain цепочку с промптом для создания краткого резюме на русском языке.

3. **Отправка**: Сформированный отчет отправляется в Telegram с ссылками на оригинальные статьи.

## Настройка

Вы можете изменить:
- Список RSS лент в переменной `AI_NEWS_RSS_FEEDS`
- Время ежедневной рассылки в методе `run()` (по умолчанию 9:00)
- Количество новостей в параметре `max_articles`
- Промпт для суммаризации в методе `setup_summarization_chain()`

## Требования

- Python 3.8+
- Токен Telegram бота
- API ключ LLM (опционально, для реальной суммаризации)
