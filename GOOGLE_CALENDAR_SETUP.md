# Настройка Google Calendar API для бота

## Шаг 1: Создание проекта в Google Cloud Console

1. Откройте [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект или выберите существующий
3. Назовите проект, например "Telegram Bot Calendar"

## Шаг 2: Включение Google Calendar API

1. В левом меню выберите **APIs & Services** → **Library**
2. Найдите "Google Calendar API"
3. Нажмите **Enable**

## Шаг 3: Создание OAuth 2.0 учетных данных

1. В левом меню выберите **APIs & Services** → **Credentials**
2. Нажмите **Create Credentials** → **OAuth client ID**
3. Если появится запрос на настройку Consent Screen:
   - Выберите тип пользователя: **External**
   - Заполните обязательные поля:
     - App name: "Telegram Bot Calendar"
     - User support email: ваш email
     - Developer contact email: ваш email
   - Нажмите **Save and Continue**
   - Пропустите раздел Scopes (нажмите **Save and Continue**)
   - Добавьте тестового пользователя (свой email) и нажмите **Save and Continue**

4. После настройки Consent Screen вернитесь к созданию OAuth client ID:
   - Application type: **Desktop app**
   - Name: "Telegram Bot Calendar Client"
   - Нажмите **Create**

5. Скачайте файл с учетными данными (кнопка **Download JSON**)
6. Переименуйте скачанный файл в `credentials.json`
7. Поместите файл `credentials.json` в ту же директорию, где находится `ai_news_agent.py`

## Шаг 4: Первая авторизация

При первом запуске бота вам потребуется пройти авторизацию:

1. Запустите бота: `python ai_news_agent.py`
2. Бот сообщит о необходимости авторизации
3. Откройте ссылку, которую предоставит бот
4. Войдите в свой Google аккаунт
5. Предоставьте разрешение на доступ к календарю
6. Скопируйте код авторизации и вставьте его в консоль
7. Файл `token.json` будет автоматически создан и сохранен для будущего использования

## Шаг 5: Использование

После настройки вы можете отправлять боту сообщения в следующем формате:

- "Создай мероприятие 'Встреча с командой' 25.12.2024 в 15:00 на 2 часа"
- "Добавь в календарь собеседование 15.01.2025 на 30 минут"
- "Мероприятие 'Презентация проекта' 20.12.2024 10:00-11:30"
- "Создай встречу с коллегами завтра в 14:00"

Бот автоматически распознает дату, время и название мероприятия, создаст его в вашем Google Календаре и отправит ссылку на мероприятие.

## Примечания

- Файл `token.json` содержит ваши учетные данные доступа. Не передавайте его третьим лицам.
- Для отзыва доступа удалите файл `token.json` и пройдите авторизацию заново.
- По умолчанию мероприятия создаются в вашем основном календаре (primary).
- Часовой пояс установлен на Europe/Moscow. При необходимости измените его в коде (класс `GoogleCalendarAssistant`, метод `create_event`).
